# JUDGE-SERVICE
## 진행 상황  
* 채점 서버에 필요한 정보들을 저장할 구조체 구현  
1) 채점 큐에서 빼낸 정보 (문제 번호, 언어, 코드, 채점 번호, 시간 제한, 메모리 제한)  
2) 채점 결과 (채점 번호, 결과, 시간, 메모리, 에러 메세지)  
3) 언어 별 설정 (컴파일 명령어, 실행 명령어, 파일 확장자, 채점 기준)
